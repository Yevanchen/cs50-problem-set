🔍 地理定位数据来源：

  • GPS信号 - 移动设备的GPS芯片
  • Wi-Fi网络 - 基于已知Wi-Fi热点位置
  • 手机基站 - 移动网络基站三角定位
  • IP地址 - 基于网络IP的大概位置

📱 获取位置信息

🔧 代码实现

// 基础用法 navigator.geolocation.getCurrentPosition( function(position) { // 成功获取位置 console.log("纬度: " + position.coords.latitude); console.log("经度: " + position.coords.longitude); }, function(error) { // 获取失败 console.log("错误: " + error.message); }, { // 选项配置 enableHighAccuracy: true, // 高精度 timeout: 10000, // 超时时间 maximumAge: 60000 // 缓存时间 } );

📊 位置对象属性详解

  • latitude - 纬度 (-90 到 90)
  • longitude - 经度 (-180 到 180)
  • accuracy - 精度（米）
  • altitude - 海拔高度（米，可能为null）
  • altitudeAccuracy - 海拔精度（米）
  • heading - 方向（度，可能为null）
  • speed - 速度（米/秒，可能为null）
  • timestamp - 获取时间戳
